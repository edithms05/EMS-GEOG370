It uses the total student population, as shown on the map behind the pie charts. I created this ratio to show the how each gender compares when it comes to school enrollment in the state of Idaho. This map represents this data fairly well, however the pie charts are hard to fully scale because of the way that Idaho's population is spread across the state.
